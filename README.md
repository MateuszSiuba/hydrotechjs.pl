# HydrotechJS

To autorska strona internetowa stworzona przy użyciu HTML, CSS i JavaScript. Strona jest hostowana na GitHub Pages z niestandardową domeną **www.hydrotechjs.pl**.

## Opis projektu

Projekt ma na celu stworzenie nowoczesnej, responsywnej i funkcjonalnej witryny internetowej. Zawiera podstawowe elementy webowe, które zapewniają przejrzysty wygląd i dobrą wydajność.

Wszystkie treści, kod źródłowy i projekt strony są chronione prawem autorskim. Kopiowanie, modyfikowanie, rozpowszechnianie lub wykorzystywanie w całości lub części bez zgody autora jest zabronione.
